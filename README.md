# 忘情茶馆 (Wangqing Teahouse)

一个基于民间爱情故事母题的互动叙事文化网站 —— 用数字绘本长卷的形式重新讲述中国民间爱情故事。

## 项目简介

本项目，我们以视觉小说为载体，以互动叙事为手段，将中国民间爱情传说进行现代化的可视化和可交互呈现。

## 技术栈

- **Vite** 6 + **React** 18
- **Tailwind CSS** 4
- **Radix UI** / **MUI** / **shadcn 风格组件**
- **TypeScript**

## 本地开发

```bash
npm install
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本（Vite build + 内联资源到单个 HTML）
npm run preview  # 预览构建结果
```
