# 忘情茶馆 - 民间爱情故事母题互动叙事网站

## 项目简介

这是一个基于设计文档 `design-spec.md` 实现的互动叙事文化网站，用数字绘本长卷的形式重新讲述中国民间爱情故事。

## 设计特点

### 视觉风格
- **新中式雅淡风格**：茶青色主色调，米宣纸背景
- **数字绘本长卷**：垂直滚动式阅读体验
- **古画静态呈现**：使用传统古画素材，克制的过渡动画

### 核心功能

1. **首页（母题图谱广场）**
   - Hero 区域展示项目介绍
   - 可玩章节卡片（白蛇传·法海篇、牛郎织女·老黄牛篇）
   - 即将开启的短篇预告
   - 爱情母题图谱展示

2. **章节阅读页**
   - 垂直长卷滚动阅读
   - 顶部薄条：返回按钮、章节名、阅读进度
   - 古画与文字交错排布
   - 交互抉择节点
   - 右下角悬浮控件

3. **交互元素**
   - 抉择拦轴：选择不同视角看故事
   - 文字段逐句淡入动画
   - 古画 Ken Burns 效果
   - 鼠标静止 1.5s 自动隐藏控件

## 技术栈

- **React 18** + **TypeScript**
- **Tailwind CSS v4** - 样式框架
- **Motion (Framer Motion)** - 动画库
- **Lucide React** - 图标库
- **Vite** - 构建工具

## 设计规范遵循

### 颜色系统
- 茶青（主色）：`#6B8775`
- 朱砂（强调）：`#B85450`
- 薄金（收集）：`#B89968`
- 米宣（背景）：`#F5EFE3`
- 纸白（表面）：`#FAF6EC`

### 字体
- 中文：Source Han Serif SC (思源宋体)
- 英文：EB Garamond

### 动效原则
- 古画进入：opacity 0→1 + scale 1.05→1.0，800ms
- 文字段：逐句 opacity 0→1 + translateY(8px)→0，300ms
- 抉择浮层：opacity 0→1 + translateY(20px)→0，400ms

## 使用说明

1. **首页浏览**
   - 查看可玩章节卡片
   - 点击"开始阅读"进入章节

2. **章节阅读**
   - 向下滚动阅读长卷
   - 遇到抉择节点时选择视角
   - 鼠标移动显示控件，静止 1.5s 自动隐藏

3. **交互功能**
   - 顶部：返回首页、查看进度
   - 右下角：目录、收集册、设置

## 设计原则

1. **尊重原典**：故事忠实于原典文献
2. **完整叙述**：每章完整讲述故事全程
3. **不重写故事**：交互只改变视角，不改变事件
4. **玩中学**：在沉浸阅读中自然吸收文化知识
