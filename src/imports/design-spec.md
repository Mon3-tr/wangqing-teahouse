## 一句话定义

**用「数字绘本长卷」语言重新讲述中国民间爱情母题的互动叙事文化网站——借鉴视觉小说的对话、抉择、视角切换交互形式，但所有交互只改变「叙事的视角/心境/氛围」，不改变事件本身；让国风青年在 15-20 分钟里，从一道可以一直下拉的古画长卷中，沉浸式重走一段千年爱情故事，并自然吸收文化知识。**

---

## Part 1 · 用研洞察

### 1.1 核心用户

**国风青年（Z 世代，18-28 岁）**：活跃于 B 站/小红书/抖音，关注汉服/古风音乐/国风插画/古典文学；偏好"传统的现代表达"；以"刷-收藏-分享"为主要内容消费模式；反感知识灌输，偏爱"原来如此"的反转叙事。

**典型用户「小棠」**：22 岁，杭州某高校汉服社成员，愿意为"做得美、做得有意思"的国风内容停留 5-15 分钟，每周分享 1-2 条传统文化内容到社交平台。

### 1.2 核心痛点

- **功能性**：想了解某个爱情故事的演变，资料散落于古籍、论文、词条，难以串联
- **体验性**：现有学术资料/百科以文字为主，缺乏视觉吸引力与情感代入
- **情感性**：年轻一代对"传统文化是父辈的"有距离感；想要的不是被教育，而是"我自己挖到了一个酷东西"的发现感与表达欲

### 1.3 价值主张

| 价值 | 用户收益 | 优先级 |
|------|---------|------|
| 参与价值 | 从旁观者变成"参与塑造视角的人" | P0 |
| 情感价值 | "原来法海/王母也有他的难处"的反差震撼 | P0 |
| 认知价值 | 在沉浸阅读中自然吸收文化知识（玩中学） | P0 |
| 收集价值 | 文化彩蛋金边收集，鼓励二刷 | P1 |
| 社交价值 | 高质量剧情卡片，可分享到小红书/朋友圈 | P1 |
| 发现价值 | 接触主流之外的冷门母题（韩凭、紫玉、阿诗玛） | P1 |

### 1.4 差异化主张

- **视角 × 反差**：玩家可以从反派（法海眼中的"妖气"实则是爱意）、物品（老黄牛旁观但不能开口）等突破性视角重看熟知故事
- **冷门 × 解谜**：把冷门母题（韩凭、紫玉与韩重）做成探案/解谜短篇，制造"揭秘感"
- **古画 × 长卷**：以传统古画为画面主体，以数字绘本长卷为呈现形态——区别于翻页式 VN，区别于纯文字百科，区别于动画视频

### 1.5 内容范围

| 优先级 | 母题 | 形式 | 比赛交付 |
|------|------|------|--------|
| ⭐⭐⭐⭐⭐ | **白蛇·法海篇** | 反派视角完整章节 | ✅ 完整 7 幕 |
| ⭐⭐⭐⭐⭐ | **牛郎织女·老黄牛篇** | 物品视角完整章节 | ✅ 时间允许时完整 |
| ⭐⭐⭐⭐ | 梁祝、韩凭夫妇、紫玉与韩重 | 短篇预告 | ✅ 预告占位 |
| ⭐⭐⭐ | 孟姜女、田螺姑娘、阿诗玛 | 母题图谱卡片 | ✅ 占位 |

## Part 2 · 交互设计

### 2.1 产品形态

- ✅ **是**：以网页为载体的互动叙事 + 文化科普作品，借鉴视觉小说的「对话框 + 插画 + 抉择」交互来增强沉浸感
- ❌ **不是**：完整的视觉小说游戏（不做存档/读档/快进/成就墙/好感度等游戏化重模块）
- ❌ **也不是**：可被用户"重写"的交互故事——所有交互只影响「叙事的呈现方式」（视角/心境/氛围），**不改变事件本身**
- 🎯 **目标**：让用户在 **15-20 分钟内**像"读了一篇会动的图文故事"那样完整体验一段民间爱情故事

### 2.2 素材形式 & 呈现形态

**古画静态呈现**：
- 画面采用传统古画素材（画像石拓片、敦煌壁画、宋元纸本设色、木版年画、连环画）
- 古画**静态呈现**为主，只做克制过渡（淡入淡出、Ken Burns、卷轴展开）
- **不做**抖动、表情切换、后退、转身等演出动作

**数字绘本长卷**：
- 章节采用**垂直长卷滚动**呈现，取古画长卷（《清明上河图》《千里江山图》）随卷轴展开阅读的意象
- 一章从头到尾就是**一道可以一直下拉的卷轴**——不是点击翻页式幻灯片
- 古画与文字交错排布，进入可视区时触发轻微动画
- 形态接近"带交互的数字互动长卷绘本"，而非传统 Galgame

### 2.3 七条创作原则（不可妥协）

1. **尊重原典**：故事主干、关键情节、结局走向必须忠实于原典文献
2. **完整叙述**：每章分段完整讲述故事全程（起因→发展→高潮→结局），不只截取抉择
3. **不重写故事**：所有事件、人物关系、结局都与原典一致，变的只是呈现方式
4. **交互服务沉浸**：用户是"选择从何种视角看故事的人"，不是"决定故事走向的人"
5. **玩中学**：关键节点附带可选的文献注解/文化背景浮窗
6. **触发"原来如此"**：通过反派视角、物品视角、内心独白等让熟知情节呈现未被讲述的一面
7. **轻量克制**：交互形式服务叙事，不堆砌游戏机制

### 2.4 关键界面

**首页（母题图谱广场）**
- Hero 区 + 当前可玩章节卡片 + 短篇预告区 + 母题图谱缩略图墙

**章节阅读页（数字绘本长卷）**
- 顶部薄条（半透明吸顶，含返回/章节名/进度%，鼠标静止 1.5s 隐藏）
- 中部卷轴主体（古画 → 文字段 → 卷上亮点 → 古画 → 卷中拦轴 → 后续段落）
- 右下角悬浮控件（自动播放/目录/收集册/设置，鼠标静止淡化）

### 2.5 长卷中的两类交互节点

**A. 卷上亮点（可点可不点，不拦截阅读）**
- 文化彩蛋亮点（古画中的物件柔光呼吸，点击弹出文化注解卡片）
- 人物头像亮点（点击查看人物背景/心声）
- 「如果当时」幻象插段（原典高潮点自动插入幻象画面，自然退出回归原典）
- 已点击亮点变金边并计入「文化收集册」

**B. 卷中拦轴（必须交互才能继续下拉）**
- 抉择拦轴（中央出现 2-3 个选项卡片，"你选择从…看"）
- 视角切换拦轴（角色头像选项，同一事件以另一视角重述）
- 信物探案拦轴（探案模式章节专属）
- 幕末拦轴（章节分幕节点）

### 2.6 十大组件

1. **卷轴文字段系统** — 角色名/旁白/心声三种样式，进入可视区逐句淡入
2. **古画场景系统** — 仿装裱画框，Ken Burns 推拉，局部高亮
3. **叙事抉择浮层** — 提示语以"你选择从…看"起头，强调视角而非走向
4. **视角切换组件** — 同一事件以另一视角重述
5. **信物探案系统** — 画面软锁定，点亮足够信物后继续
6. **「如果当时」幻象浮窗** — 自动出现非用户选择，幻象自然消散，原典走向不变
7. **文化彩蛋 / 文献注解系统** — 浮窗 + 收集册 + 金边
8. **多结局收集墙** — 收集册形式，呈现已读视角与彩蛋
9. **剧情卡片分享生成器** — 古画 + 关键台词 + 章节名 + 二维码
10. **顶部薄条 + 右下角悬浮控件** — 自动播放、目录、收集册、设置、退出

## Part 3 · 视觉设计

### 3.1 视觉风格

**关键词**：新中式雅淡 / 茶汤静青 / 纸本质感 / 卷轴长卷 / 文气克制

**整体基调**：明亮（带一层薄雾感）/ 暖米底 + 茶青点睛 / 雅而暖、文气却不老气

**参考产品**：《中国奇谭》《长安三万里》、故宫文创新品、《单读》杂志装帧、《大鱼海棠》动画美术

### 3.2 核心视觉变量

```css
/* 品牌色 */
--color-primary:        #6B8775;  /* 茶青 · 主色 */
--color-primary-light:  #869B8A;  /* 悬停 */
--color-primary-dark:   #566F60;  /* 强调 */

/* 强调辅助色（克制使用，全篇 < 8%） */
--color-accent-cinnabar: #B85450;  /* 朱砂 · 卷中拦轴/收藏 */
--color-accent-gold:     #B89968;  /* 薄金 · 已收集金边 */

/* 空间色（卷轴底色） */
--color-bg:        #F5EFE3;  /* 米宣 · 仿宣纸 */
--color-surface:   #FAF6EC;  /* 纸白 */
--color-overlay:   rgba(245,239,227,0.85);  /* 薄雾 */
--color-border:    #D9D0BE;  /* 淡墨 */

/* 文字色 */
--color-text:        #2C2A26;  /* 古墨 */
--color-text-muted:  #6E665A;  /* 远墨 · 旁白 */
--color-text-faint:  #8C8474;  /* 淡墨 · 心声 */

/* 字体 */
--font-serif:    'Source Han Serif SC', 'Noto Serif CJK SC', serif;
--font-en-serif: 'EB Garamond', Georgia, serif;

/* 字号（基于 17px 正文，长卷阅读放大） */
--text-display:  40px;  --text-title:    28px;
--text-heading:  22px;  --text-body:     17px;
--text-narration: 18px; --text-inner:    16px;

/* 行高 */
--leading-body:      1.85;  /* 正文长卷阅读 */
--leading-narration: 2.0;   /* 旁白留白感 */

/* 间距（4px 基础，长卷阅读适当放大） */
--space-xl:   40px;   /* 古画与文字段之间 */
--space-2xl:  64px;   /* 幕与幕之间 */
--space-3xl:  120px;  /* 章与章之间 */

/* 圆角（小圆角 · 古雅克制） */
--radius-md:  6px;   /* 按钮、抉择卡片 */

/* 容器宽度 */
--container-reading:  680px;  /* 卷轴文字最大宽度 */
--container-painting: 880px;  /* 古画最大显示宽度 */

/* 动效 */
--duration-fast:   200ms;   --duration-normal: 400ms;
--duration-slow:   800ms;   --duration-epic:   1600ms;
--ease-out:    cubic-bezier(0, 0, 0.2, 1);
--ease-spring: cubic-bezier(0.16, 1, 0.3, 1);
```

### 3.3 关键搭配

| 场景 | 配色组合 |
|------|---------|
| 主阅读页背景 | 米宣 #F5EFE3 + 古墨字 #2C2A26 |
| 主按钮 | 茶青底 + 米宣文字 |
| 抉择卡片 | 纸白半透明底 + 顶部朱砂细线 + 古墨字 |
| 古画装裱 | 纸白底 + 1px 淡墨边 + 极淡阴影 |
| 已收集彩蛋 | 1px 薄金边 + 金色淡发光 |
| 顶部薄条 | 薄雾色 + backdrop-blur(12px) |

### 3.4 动效原则

- ✅ 古画进入：opacity 0→1 + scale 1.05→1.0（Ken Burns 收缩感），800ms
- ✅ 文字段：每句 opacity 0→1 + translateY(8px)→0，300ms
- ✅ 抉择浮层：opacity 0→1 + translateY(20px)→0 + backdrop-blur 0→12px，400ms
- ✅ 幻象消散：opacity 1→0 + filter blur(0)→blur(8px)，1600ms
- ❌ 禁止：弹跳、抖动、震动、晃动、玻璃拟态、霓虹光、粒子动效、视差滚动、3D 倾斜

### 3.5 视觉禁忌

- ❌ 纯黑 #000、纯白 #FFF
- ❌ 饱和度 > 70% 的鲜艳色
- ❌ 蓝色（与中式色谱冲突，除非"夜读模式"）
- ❌ 玻璃拟态、霓虹光、粒子动效、星空背景、3D 倾斜卡片
